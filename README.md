# Seal Point — Cinematic Portfolio

Award-winning-style scroll-driven portfolio site. A gold Seal Point coin travels
through three sequential AI-generated clips (arrival → construction → delivery),
scrubbed as canvas frame sequences in one continuous scroll timeline.

## Structure

```
site/
  index.html          single page
  css/main.css        design system (Void/Structure/Fog/Gold, Anton + Inter)
  js/manifest.js      frame-sequence manifest (regenerated by tools/extract-frames.sh)
  js/main.js          scroll engine: Lenis + GSAP ScrollTrigger + canvas frame player
  vendor/             gsap, ScrollTrigger, lenis (vendored, no CDN at runtime)
  fonts/              Anton (display), Inter (body) — self-hosted woff2
  assets/frames/clipN f_0001.webp… scrub frames, 12fps, 1920w
  assets/img/coin.png transparent coin render (UI seal glyph, favicon, footer)
tools/
  extract-frames.sh   MP4 → webp frame sequences + manifest regen
seal-point-coin.glb   source 3D coin model
```

## Run

```
node <any static server> site/  # e.g. npx serve site
```

The dev server used during the build: see scratchpad `site-server.js` (port 8080).

## Design rules

- Colors: Void `#060607` (bg) · Structure `#261813` (borders/cards) · Fog `#9CA3A8`
  (body text) · Gold `#D4A24C` (ALL interactivity — nothing else signals clickable)
  · Ice `#C7ECFA` reserved for in-video moments only, never flat UI.
- Type: Anton condensed display for headings (kinetic char reveals), Inter body.
- Site-wide grain overlay matches the video grain.
- Scroll-seal progress glyph bottom-right: ghost outline → gold fill → stamp punch at 100%.

## Frame pipeline

Clips are generated on Higgsfield (Seedance 2.0, std, 1080p, 16:9, 8s, silent,
coin photo passed as identity reference on every generation), then:

```
tools/extract-frames.sh clip1.mp4 clip2.mp4 clip3.mp4
```

extracts 12fps webp sequences and rewrites `site/js/manifest.js` with real counts.
The scroll engine adapts to whatever counts the manifest declares.
